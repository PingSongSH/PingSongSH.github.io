Microsoft Support Tool
Introduction
50,000+ Support Agents for Windows Store, Microsoft Store, Windows Phone, Microsoft Azure, Office365, XBOX etc.. will use the support tool( code name CMAT) to help Microsoft end customers solve their billing/subscription problems in a time pressure.
Chanllenge
30+ business groups will be the partners providing requirements and sign off design proposals; The population of users are huge and the business style are total different. We always heard complaints from users and their mangers about the performance, so productivity is king;
Achievement
Streamline workflow/zero window jump/be relevant and contextual improved the performance greatly. we made sure simplicity and consistency through the entire tool design and partner team onboard the tool ahead of planed schedule which is a big coat saving to get rid of the old version.

We adopted user centered design approach to kick off the project and defined the prioritized design principles based on user interview.
